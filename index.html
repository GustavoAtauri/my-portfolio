Projeto Hospital

Este projeto fictício consiste no desenvolvimento de uma plataforma institucional para o "Hospital Central", focada em otimizar a jornada do paciente através de uma interface intuitiva e funcional. O objetivo principal foi centralizar informações essenciais — como especialidades, corpo clínico e infraestrutura — garantindo que o usuário encontre o que precisa com agilidade. Além da área informativa, o sistema conta com um módulo dedicado ao agendamento de consultas, priorizando a clareza visual e a facilidade de navegação para proporcionar uma experiência segura e profissional no ambiente de saúde.
